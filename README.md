# mp3-tag-editor(C)
A simple C-based application to read and edit MP3 file metadata such as title, artist, and album using ID3 tags and file handling concepts.

## 🔧 Features
- Read MP3 tag details
- Edit title, artist, and album
- Works using ID3 tag format
- Uses file handling and binary data processing

## 💻 Technologies Used
- C Programming
- File Handling
- Binary File Processing

## 📚 What I Learned
- How MP3 files store metadata (ID3 tags)
- Working with binary files in C
- Reading and modifying structured data in files
